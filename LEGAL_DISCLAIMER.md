## Kali Linux Tools

KaliGuard AI leverages tools bundled with Kali Linux, including but not limited to:
- **nmap**, **masscan** (network scanning)
- **nikto**, **sqlmap**, **burpsuite** (web vulnerability scanning)
- **hashcat**, **john**, **hydra** (password analysis)
- **wireshark**, **tcpdump**, **snort** (network monitoring)
- **aircrack-ng**, **wifite**, **kismet** (wireless security)
- **volatility**, **binwalk**, **yara** (forensics and malware analysis)
- **metasploit-framework** (exploitation framework)
- **ghidra**, **radare2** (reverse engineering)

All these tools are intended for **ethical, authorized use only**.

## Allowed Networks (Default Configuration)

By default, KaliGuard AI is configured to operate only on **private IP ranges**:
- `192.168.0.0/16` (Class C private)
- `10.0.0.0/8` (Class A private)
- `172.16.0.0/12` (Class B private)

**Do NOT modify these settings to target public IPs unless you have explicit authorization.**
